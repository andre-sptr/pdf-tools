Sebuah aplikasi web komprehensif yang dirancang untuk mempermudah pengelolaan dan manipulasi file PDF. Aplikasi ini menyediakan antarmuka pengguna yang bersih dan modern untuk melakukan berbagai operasi PDF langsung dari browser Anda.

## âœ¨ Fitur Utama

Berdasarkan komponen yang tersedia, aplikasi ini memiliki beberapa alat utama:
- **Merge PDF**: Menggabungkan beberapa file PDF menjadi satu dokumen.
- **Split PDF**: Memisahkan satu file PDF menjadi beberapa halaman atau dokumen yang lebih kecil.
- **Compress PDF**: Mengurangi ukuran file PDF tanpa mengorbankan kualitas secara signifikan.
- **Convert To PDF**: Mengubah format file lain (seperti gambar) menjadi format PDF.
- **Convert From PDF**: Mengubah file PDF menjadi format lain.
- **Rotate PDF**: Memutar orientasi halaman di dalam file PDF.
